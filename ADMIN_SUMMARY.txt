Storage Location:
  → Supabase Cloud Storage
  → Bucket: product-images
  → Public access enabled

Upload Process:
  1. Select image from your computer
  2. Click "Upload Image" button
  3. Wait for confirmation
  4. Image URL auto-saved to database
  5. Image appears in store automatically

Technical Details:
  → Max file size: 5MB
  → Formats: JPG, PNG, GIF, WebP
  → Public URLs auto-generated
  → URLs stored in products.image_url column

═══════════════════════════════════════════════════════════════
  DATABASE STRUCTURE
═══════════════════════════════════════════════════════════════

Table: products
  - id (auto-increment)
  - name
  - category
  - price
  - image_url (public URL to Supabase Storage)
  - created_at
  - updated_at

Table: admin_users
  - id (UUID)
  - email
  - created_at

Storage: product-images (bucket)
  - products/[timestamp]_[random].[ext]
